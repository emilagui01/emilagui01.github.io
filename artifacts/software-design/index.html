Artifact: Travlr Full Stack Application

The Travlr Full Stack Application is a MEAN stack travel management application developed using MongoDB, Express, Angular, and Node.js. This artifact demonstrates frontend and backend integration, reusable Angular components, RESTful API communication, and JWT-based authentication.
